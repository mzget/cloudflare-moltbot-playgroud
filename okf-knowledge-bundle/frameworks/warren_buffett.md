# Warren Buffett — ขอบเขตความรู้ (Circle of Competence) คูเมืองทางเศรษฐกิจ (Moat) และมูลค่าที่แท้จริง

## 1. ขอบเขตความรู้ (Circle of Competence)

ลงทุนเฉพาะในธุรกิจที่เราเข้าใจอย่างถ่องแท้เท่านั้น

## 2. คูเมืองทางเศรษฐกิจ (Economic Moat)

มองหาบริษัทที่มีความได้เปรียบในการแข่งขันที่ยั่งยืนยาวนาน เช่น:
- แบรนด์ที่แข็งแกร่ง
- ต้นทุนในการเปลี่ยนต่ำ (Switching Costs)
- ความได้เปรียบด้านต้นทุน (Cost Advantage)

## 3. กำไรของเจ้าของ (Owner's Earnings)

โฟกัสกระแสเงินสดที่แท้จริงที่บริษัทสร้างขึ้นสำหรับเจ้าของ (ไม่ใช่เพียงกำไรทางบัญชี)

```
Owner's Earnings = กำไรสุทธิ + ค่าเสื่อมราคา/ค่าตัดจำหน่าย - Maintenance CapEx
```

## 4. ส่วนเผื่อเพื่อความปลอดภัย (Margin of Safety)

ซื้อหุ้นในราคาที่ต่ำกว่ามูลค่าที่แท้จริงอย่างมีนัยสำคัญ (ส่วนลด 20–30% ขึ้นไป) เพื่อป้องกันความผิดพลาดและรองรับความไม่แน่นอน

## 5. เน้นคุณภาพเหนือราคาถูก

ซื้อบริษัทที่ยอดเยี่ยมในราคาที่เหมาะสม ดีกว่าซื้อบริษัทที่ธรรมดาในราคาที่ถูกมาก
