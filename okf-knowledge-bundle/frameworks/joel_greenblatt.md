# Joel Greenblatt — สูตรมหัศจรรย์ (Magic Formula) คุณภาพบวกความถูก

สูตรมหัศจรรย์ (Magic Formula) มุ่งเน้นการหาบริษัทที่ดีในราคาที่ถูกอย่างเป็นระบบ

## 1. อัตราผลตอบแทนต่อเงินทุน (Return on Capital — ROC)

วัดคุณภาพของธุรกิจ:

```
ROC = EBIT ÷ (สินทรัพย์ถาวรสุทธิ + เงินทุนหมุนเวียนสุทธิ)
```

## 2. อัตราผลตอบแทนจากการดำเนินงาน (Earnings Yield — EY)

วัดความถูกของหุ้น:

```
Earnings Yield = EBIT ÷ Enterprise Value (EV)
EV = มูลค่าตลาด + หนี้สิน - เงินสด
```

## 3. การจัดลำดับคะแนนรวม (Combined Rank)

นำบริษัทมาจัดอันดับ ROC และ EY แล้วรวมลำดับเข้าด้วยกัน บริษัทที่มีผลรวมลำดับน้อยที่สุดถือเป็นบริษัทที่น่าสนใจที่สุด

## 4. การหลีกเลี่ยงกับดักมูลค่า (Value Traps)

ตรวจสอบว่าเหตุใดหุ้นจึงมีราคาถูกผิดปกติ เช่น:
- กำไรที่เกิดจากรายการพิเศษครั้งเดียว
- อุตสาหกรรมกำลังจะล่มสลายเชิงโครงสร้าง
